# Programadores para la Paz – Mini-sitio comunitario

## Proyecto único de Semana 5

Este repositorio se trabajará desde la Clase 21 hasta la Clase 25.

## Actividad Clase 21

Inicio del mini-sitio comunitario que consume una API con async y await.

## Objetivo del proyecto

Construir progresivamente un mini-sitio comunitario que permita consultar mensajes, recursos y piezas de un calendario editorial mediante una API desarrollada con Node.js y Express.

## Enfoque ciudadano

El proyecto trabaja difusión responsable, calendario editorial, consistencia comunicativa, revisión de fuentes, moderación editorial básica y lenguaje responsable y no estigmatizante.

## Flujo de ramas

main:
Rama base actualizada por docentes.

nombres_apellidos:
Rama personal de cada estudiante.

No trabajes directamente sobre main.

```bash
